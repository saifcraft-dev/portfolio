You are a senior UI/UX designer specializing in modern SaaS and developer 
portfolio websites. I need you to redesign the color palette for my freelance 
fullstack developer portfolio website called "DevStudio."

=== PROJECT CONTEXT ===
- Site: DevStudio — a freelance portfolio & service platform
- Owner: Saif Khan (Freelance Fullstack Developer)
- Audience: Startup founders, small business owners, and tech clients looking 
  to hire a developer for landing pages, business sites, custom web apps, 
  and AI features ($250–$2,500+ projects)
- Vibe goal: Modern, premium, trustworthy, tech-forward, clean, and 
  conversion-focused (must inspire confidence to pay for services)

=== CURRENT PALETTE (needs improvement) ===
- Background: #fafaf8 (light gray)
- Foreground: #212121 (near black)
- Primary: #2563eb (blue)
- Secondary: #0891b2 (teal)
- Accent: #a855f7 (purple)
- CTA: #ea580c (amber/orange)
- Footer: #1e3a5f (dark blue)
- Plus: muted, border, destructive tokens

=== PROBLEMS WITH CURRENT PALETTE ===
1. Too many competing accent colors (blue + teal + purple + amber) — feels 
   chaotic and unfocused
2. The CTA hex (#ea580c) doesn't match its HSL (38 92% 50%) — inconsistency
3. Generic "bootstrap blue" primary lacks personality
4. No dark mode variant defined
5. Doesn't feel distinctive or premium for a developer brand

=== WHAT I WANT YOU TO DO ===
Provide TWO complete color palette options (Option A and Option B) with 
DIFFERENT moods:
  • Option A: Sleek & premium (think Linear, Vercel, Stripe aesthetic)
  • Option B: Bold & creative (think Framer, Raycast, modern dev tools)

For EACH option, deliver:

1. A FULL token list with:
   - Token name | HSL value | Hex | Purpose/usage
   - Include: background, foreground, primary, primary-foreground, secondary, 
     accent, muted, muted-foreground, border, input, ring, destructive, 
     success, warning, cta, footer, card, card-foreground, popover

2. BOTH light mode AND dark mode variants (matching token names)

3. A reasoning paragraph explaining:
   - Why this palette fits a freelance developer brand
   - The emotional/psychological intent of each main color
   - How it improves conversion on pricing tiers and CTAs

4. Accessibility notes:
   - WCAG AA contrast ratios for text-on-background combos
   - Confirm CTA buttons meet 4.5:1 minimum

5. Usage guidance:
   - Which color for hero gradients
   - Which for pricing card highlights (especially the "popular" tier)
   - Which for trust signals (testimonials, tech stack icons)
   - Footer treatment

6. Recommended GRADIENTS (2–3 per palette) using the new tokens, with CSS

7. Tailwind CSS v4 / shadcn-ui compatible CSS variables block ready to paste 
   into globals.css (using oklch() or hsl() format)

=== CONSTRAINTS ===
- Keep typography (Space Grotesk + Outfit) and 0.75rem border radius
- Maintain high contrast for readability
- Limit to ONE primary brand color + ONE accent (no more 4-color rainbows)
- Must look professional enough to charge $2,500+ for projects
- Should differentiate from typical "blue SaaS" templates

=== OUTPUT FORMAT ===
Use clean tables for token lists, code blocks for CSS, and short paragraphs 
for reasoning. No fluff.
